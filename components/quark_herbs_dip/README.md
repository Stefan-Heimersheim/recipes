# Herb Quark Dip

## Ingredients

* 250g quark (low-fat works)
* Some milk, cream, or plain yogurt
* Garlic
* Salt, (pepper)
* 1x salad seasoning with spring herbs
* Maggi seasoning No. 4
* Chives (preferably fresh)

## Instructions

* Mix everything together
* Add milk/cream/yogurt as needed, quite generously (e.g. half a cup of yogurt)
